# Fibonacci Sequence #

Calculate the values of the Fibonacci Sequence and display them on the OUT register.

## Code ##

    Addr    Value  Opcode
    0x0: 	0x71   ldi  0x1
    0x1: 	0x4E   sta  [0xE] 
    0x2: 	0x70   ldi  0x0
    0x3: 	0xE0   out
    0x4: 	0x2E   add  [0xE] 
    0x5: 	0x4F   sta  [0xF]
    0x6: 	0x1E   lda  [0xE]
    0x7: 	0x4D   sta  [0xD]
    0x8: 	0x1F   lda  [0xF]
    0x9: 	0x4E   sta  [0xE]
    0xA: 	0x1D   lda  [0xD]
    0xB: 	0x80   jc   0x0
    0xC: 	0x63   jmp  0x3
    0xD: 	0x00   0x00
    0xE: 	0x00   0x00
    0xF: 	0x00   0x00
